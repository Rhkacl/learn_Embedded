비트 마스킹
비트를 제어할 때
원하는 위치에 1을 넣고 싶으면
원하는 위치만큼 1을 시프트 시키고 or 연산을 실행
ex)
PORTA의 4번에 1을 넣고 싶으면 
PROTA |= (1<<4);    // 이렇게 함

원하는 위치에 0을 넣고 싶으면
원하는 위치만큼 1을 시프트 시코고 NOT 연산을 걸어서 원하는 위치를 0으로 만듦
-> 기존 PORT와 AND 연산을 함 
ex)
PORTA의 3번에 0을 넣고 싶으면
PORTA &= ~(1<<3);   // 이렇게 함

원하는 위치에 0인지?1인지? 확인하는 방법은 원하는 위치에 1을 시프트 시켜서 포트와 and연산을 함
ex)
PORTA 의 2번에 있는 값을 알고 싶다면 
PORTA &= (1<<2);를 진행하면 됨
